TRABALHO DE IMPLEMENTAÇÃO ANALISADOR LÉXICO

“endfor”

Excluir a marca:
“;”

Alterar as marcas:
“end” para “endif”.

Identificadores devem ser iniciados com uma letra concatenada a uma sequência de zero ou mais letras, números ou underline.
